# BlockCraft Lite

Mini jeu voxel original en WebGL, intégré à Branlly. Le rendu procédural fonctionne hors ligne avec génération de terrain, arbres, eau fluide, minerais cristal et textures procédurales par bloc.

- ZQSD/WASD : déplacement
- Souris : regarder
- Espace : sauter
- Clic gauche/droit : retirer/poser
- 1–7 ou molette : sélectionner un bloc ou l'eau
- Shift : courir
- F : activer/désactiver le mode vol
- Ctrl ou C : descendre en mode vol
- R : générer un nouveau monde
- Échap : pause
